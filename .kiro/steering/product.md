# Product Overview

家計簿アプリ「kakeibo-antigrav」は、モバイルに近い直感的な操作感で家計を管理できるPWAです。主に日本国内の個人ユーザーを対象に、収入・支出の入力、カテゴリ別分析、CSVインポート/エクスポート、検索機能などを提供します。

## Core Capabilities
- モバイルライクなテンキー入力で迅速に明細を追加・編集
- カレンダー表示で月ごとの合計と日別トランザクションをアコーディオンで閲覧
- グラフ（棒グラフ・円グラフ）によるカテゴリ別内訳表示
- キーワード検索と検索結果のサマリー計算
- CSVインポート/エクスポートによるデータ移行とバックアップ

## Target Use Cases
- 日常的に支出・収入を入力したい個人
- 1か月の支出をカテゴリ別に可視化したいユーザー
- CSVで保存したデータを別アプリへ移行したいユーザー
- モバイルデバイスで手軽に家計を管理したい人

## Value Proposition
- PWAでインストール不要、オフライン時も利用可能
- SvelteKitとTypeScriptを採用し高速で型安全な開発体験を実現
- カスタマイズ可能なカテゴリやレシート管理で個人に合わせた操作感を提供
